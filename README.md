# Vibe Coding Mastery — Slide Deck

A cinematic, AQaddoura-style (sci-fi / HUD) presentation deck for the 3-hour
**Vibe Coding Mastery** session by Ahmed Qaddoura. Built with [reveal.js](https://revealjs.com).

**▶ Present:** https://aqaddora.github.io/vibe-coding-mastery-deck/

## Presenter controls

| Key | Action |
|---|---|
| `→` / `Space` | Next slide / fragment |
| `←` | Previous |
| `F` | Fullscreen |
| `S` | Speaker view (timer + notes) |
| `O` / `Esc` | Slide overview |
| `B` | Black out the screen |
| `?` | All shortcuts |

Each slide deep-links via the URL hash (e.g. `#/9`), so you can jump straight to any slide.

## Edit

Everything lives in a single self-contained `index.html` (theme + slides + config).
Open it, edit the `<section>` blocks, refresh. No build step.

Source outline: the session `flow.md` / `script.md` / `slides.md` (private).
